# boroactions/lisp-scripting@v0

This GitHub action allows using Common Lisp in shell scripts with shebang:
```sh
#!/usr/bin/env lisps

(print 'hello)
```

See [borodust/lisp-scripts](https://github.com/borodust/lisp-scripts/lisps/) for further details.
